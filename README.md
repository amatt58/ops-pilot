# OpsPilot

An AI-augmented internal operations platform for support and ops teams. Agents use OpsPilot to triage, understand, and resolve tickets faster — assisted by AI summarisation, response drafting, and semantic retrieval.

Built as a portfolio project to demonstrate production-quality full-stack TypeScript engineering, practical AI integration, and modern operational tooling patterns.

---

## What it does

- **Ticket inbox** — triage and manage support tickets with filtering, sorting, and priority management
- **AI summarisation** — generate structured summaries of ticket threads in one click
- **AI response drafting** — draft customer replies using ticket context and knowledge base retrieval
- **Semantic search** — find similar resolved tickets and relevant knowledge articles using vector similarity
- **Activity timeline** — immutable audit trail of every action taken on a ticket, including AI actions
- **Knowledge base** — searchable internal reference articles surfaced during response drafting
